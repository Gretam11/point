Point grid
Grid
This grid is implemented as an html table. It shows paint spread by means of different functions. You can start paint reaction just by clicking on any cell. You can clear the grid with ESC button.
Grid settings
Grid settings update requires grid reload.
Angular Grid Engine - renders grid using angular framework's NgForOf structural directives with NgClass directive for every table cell. It also relies on angular bindings and change detection mechanism.
Custom Grid Engine - renders grid using almost native JS dom api (via Renderer2) to create, remove and update dom elements according to changes in JS object model.
Paint settings
Paint settings may be updated in real time during paint spread.
FPS meter
FPS meter is allocated at the bottom panel.
